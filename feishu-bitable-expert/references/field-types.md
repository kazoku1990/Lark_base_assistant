## 概览
飞书多维表格提供多种字段类型，每种类型适用于不同的数据存储和展示需求。合理选择字段类型是构建高效数据结构的基础。

## 基础字段类型

### 文本
- **用途**：存储短文本、名称、描述等
- **特性**：
  - 支持最多5000字符
  - 可设置唯一值验证
  - 支持正则表达式格式验证
- **适用场景**：姓名、标题、备注、编号等

### 数字
- **用途**：存储数值、金额、百分比等
- **特性**：
  - 支持整数和小数
  - 可设置小数位数
  - 可设置最小值和最大值
  - 支持千分位显示
- **适用场景**：金额、数量、评分、年龄等

### 单选
- **用途**：从预设选项中选择一个值
- **特性**：
  - 选项可设置颜色标记
  - 可设置默认值
  - 支持动态配置选项列表
- **适用场景**：状态、优先级、分类、阶段等

### 多选
- **用途**：从预设选项中选择多个值
- **特性**：
  - 选项顺序可自定义
  - 支持选项分组
  - 可设置最大选择数量
- **适用场景**：标签、技能、多分类、相关项目等

### 日期
- **用途**：存储日期或日期时间
- **特性**：
  - 可配置为仅日期或日期时间
  - 可设置日期范围
  - 支持时区设置
  - 可显示相对时间（如"3天前"）
- **适用场景**：截止日期、开始时间、创建时间、生日等

### 附件
- **用途**：上传和管理文件、图片等
- **特性**：
  - 支持多种文件格式
  - 可设置文件大小限制
  - 支持预览和在线编辑
  - 可设置附件数量限制
- **适用场景**：合同扫描件、设计稿、发票、头像等

### 人员
- **用途**：关联飞书用户
- **特性**：
  - 支持单选和多选
  - 自动同步飞书组织架构
  - 支持@提醒
  - 可根据部门筛选
- **适用场景**：负责人、参与者、审批人等

### 复选框
- **用途**：标记是/否状态
- **特性**：
  - 二元状态（已勾选/未勾选）
  - 可设置默认值
- **适用场景**：完成状态、激活状态、确认标志等

### 电话
- **用途**：存储电话号码
- **特性**：
  - 自动格式化
  - 支持点击拨号
- **适用场景**：手机号、座机等

### 邮箱
- **用途**：存储邮箱地址
- **特性**：
  - 格式验证
  - 支持点击发送邮件
- **适用场景**：工作邮箱、联系人邮箱等

### URL
- **用途**：存储网页链接
- **特性**：
  - 格式验证
  - 支持点击跳转
  - 可显示标题或完整URL
- **适用场景**：官网链接、文档地址、资源链接等

## 高级字段类型

### 进度
- **用途**：显示0-100%的进度
- **特性**：
  - 可视化进度条
  - 支持手动输入或公式计算
- **适用场景**：任务完成度、项目进度等

### 评分
- **用途**：星级评分
- **特性**：
  - 自定义星级数量（1-10星）
  - 支持半星评分
  - 可显示平均分
- **适用场景**：满意度评分、质量评分、绩效评分等

### 货币
- **用途**：存储货币金额
- **特性**：
  - 自动格式化为货币样式
  - 支持多币种
  - 可设置货币符号位置
- **适用场景**：预算、收入、支出、报价等

### 公式
- **用途**：基于其他字段计算值
- **特性**：
  - 支持丰富函数库
  - 实时自动计算
  - 可引用其他表的数据
- **适用场景**：自动计算字段、数据验证、条件判断等

### 查找引用
- **用途**：从关联表中获取数据
- **特性**：
  - 动态获取关联表的字段值
  - 支持多个关联记录
  - 数据自动同步
- **适用场景**：获取关联项目的信息、汇总子表数据等

### 汇总
- **用途**：汇总关联表的数据
- **特性**：
  - 支持多种汇总方式（计数、求和、平均值等）
  - 基于条件汇总
  - 实时计算
- **适用场景**：统计任务数量、汇总项目预算、计算平均评分等

### 成员
- **用途**：指定表格的协作成员
- **特性**：
  - 区分字段权限和视图权限
  - 支持批量分配
  - 可继承表级权限
- **适用场景**：指定表格管理员、协作者等

### 创建时间
- **用途**：自动记录记录创建时间
- **特性**：
  - 系统自动生成，不可手动修改
  - 精确到秒
- **适用场景**：追踪记录创建时间、排序等

### 修改时间
- **用途**：自动记录记录最后修改时间
- **特性**：
  - 系统自动更新
  - 可用于监控数据变更
- **适用场景**：数据审计、筛选最近更新的记录等

### 创建人
- **用途**：自动记录记录创建者
- **特性**：
  - 自动关联飞书用户
  - 不可手动修改
- **适用场景**：责任追溯、按创建人筛选等

### 修改人
- **用途**：自动记录最后修改者
- **特性**：
  - 自动更新
  - 可用于责任追溯
- **适用场景**：数据审计、查看谁最后修改了记录等

### 自动编号
- **用途**：为每条记录自动生成唯一编号
- **特性**：
  - 支持自定义前缀、后缀
  - 可设置数字位数
  - 自动递增
- **适用场景**：订单号、工单号、序列号等

### 地理位置字段
- **用途**：存储地图位置信息
- **特性**：
  - 支持搜索地名和地址
  - 在地图上标注位置
  - 计算距离
- **适用场景**：客户地址、门店位置、送货地址等

### 签名
- **用途**：手写签名
- **特性**：
  - 支持触屏手写
  - 图片格式存储
- **适用场景**：审批签名、确认签名、验收签名等

## 关联字段

### 双向关联
- **用途**：建立两个表之间的双向关系
- **特性**：
  - 在两个表中自动创建关联字段
  - 一方修改，另一方同步更新
  - 可设置关联记录数量限制（1对1、1对多、多对多）
- **适用场景**：项目与任务、客户与订单、部门与员工等

### 配置要点
1. **选择关联表**：建立表之间的引用关系
2. **显示字段**：选择在关联字段中显示的目标表字段
3. **关联方式**：
   - 1对1：一条记录只能关联一条记录
   - 1对多：一条记录可以关联多条记录
   - 多对多：双向都可以关联多条记录
4. **条件过滤**：设置关联时的过滤条件

## 字段配置最佳实践

### 命名规范
- 使用清晰、简短的中文或英文命名
- 字段名应该能准确描述字段内容
- 避免使用缩写（除非是公认的缩写）

### 字段顺序
- 重要字段靠前
- 逻辑相关的字段放在一起
- 便于数据录入和查看

### 字段类型选择
- 优先使用基础字段类型
- 避免过度使用复杂字段类型
- 考虑数据扩展性

### 数据验证
- 为关键字段设置验证规则
- 使用唯一值字段防止重复
- 正则表达式验证格式

### 默认值
- 为常用字段设置合理的默认值
- 减少手动录入工作量
- 提高数据一致性

## 常见场景示例

### 场景1：项目管理表字段设计
```yaml
基础表结构：
- 项目名称（文本）
- 项目编号（自动编号：PROJ-001）
- 项目负责人（人员）
- 项目状态（单选：未开始/进行中/已完成/已暂停）
- 项目优先级（单选：高/中/低）
- 开始日期（日期）
- 截止日期（日期）
- 完成进度（进度）
- 项目预算（货币）
- 实际花费（货币）
- 项目描述（多行文本）
- 项目附件（附件）
- 创建时间（创建时间）
- 修改时间（修改时间）

关联表：
- 任务表（通过关联字段链接）
```

### 场景2：客户管理表字段设计
```yaml
客户表：
- 客户名称（文本，唯一值）
- 客户编号（自动编号：CUST-001）
- 客户类型（单选：潜在客户/意向客户/成交客户）
- 客户来源（单选：官网推荐/转介绍/市场活动/其他）
- 客户行业（多选）
- 客户规模（单选：初创企业/中小企业/大型企业）
- 联系人姓名（文本）
- 联系电话（电话）
- 联系邮箱（邮箱）
- 客户地址（地理位置）
- 客户价值（评分：1-5星）
- 跟进状态（单选：待跟进/跟进中/已成交/已流失）
- 下次跟进时间（日期）
- 客户备注（多行文本）

关联表：
- 联系记录表（记录每次跟进详情）
- 商机表（跟踪销售机会）
```

### 场景3：库存管理表字段设计
```yaml
商品表：
- 商品名称（文本）
- 商品编号（自动编号：SKU-001）
- 商品分类（单选）
- 商品品牌（单选）
- 库存数量（数字）
- 安全库存（数字）
- 库存单位（单选：个/箱/件/套）
- 成本价（货币）
- 销售价（货币）
- 供应商（关联供应商表）
- 仓库位置（单选）
- 商品图片（附件）
- 商品状态（公式：库存数量<=安全库存?"库存不足":"正常"）

库存变动表（关联商品表）：
- 变动类型（单选：入库/出库/调拨/盘点）
- 变动数量（数字）
- 变动时间（日期时间）
- 经办人（人员）
- 变动原因（文本）
- 单据号（文本）
```
