# Lark Base Assistant (飞书多维表格助手)

飞书多维表格设计专家，提供从字段定义到视图搭建的全流程指导。

## 功能特性

本Skill封装了飞书多维表格的官方文档知识，帮助用户：

- **数据结构设计**：指导字段类型选择、关联关系配置
- **视图规划与搭建**：支持表格、看板、日历、甘特图等7种视图
- **公式与函数编写**：提供60+常用函数和复杂公式案例
- **自动化规则配置**：设计触发器、条件和动作
- **高级功能配置**：权限管理、工作流、仪表盘、应用模式

## 使用方法

### 方法1：直接使用打包好的Skill文件（推荐）

1. 在Coze中创建Bot或Skill
2. 下载 `feishu-bitable-expert.skill` 文件
3. 在Coze中导入该文件
4. 根据Bot的触发语境，智能体会自动引用相关参考文档

## 实现方式

本Skill完全基于**自然语言指导**实现，无需脚本。

### 为什么不用脚本？
- ✅ 内容创作/知识咨询：由智能体完成设计指导和最佳实践建议
- ✅ 结构化推理：根据业务场景推荐最优方案
- ✅ 逻辑分析：公式编写、自动化规则设计
- ✅ 无技术障碍：所有功能都是指导类任务，智能体完全胜任

### 智能体在Skill中的作用
1. 需求理解：理解用户的业务场景和需求
2. 结构设计：推荐合适的字段类型和视图配置
3. 方案生成：根据场景提供定制化的设计建议
4. 最佳实践：基于官方文档提供权威的实践指导
5. 问题解答：解答用户在使用过程中的问题
